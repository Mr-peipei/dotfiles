## Scripts Installation

1. Install the git from package manager

```bash
sudo apt install git
```


2. Run a shell script

```bash
git clone git@github.com:Mr-peipei/dotfiles.git
cd dotfiles
bash ./scripts/dotfiles-install.sh $ROOT_PASSWORD
```
